# Examples

Common prompts and workflows for Fastmail Courier.

## Email Prompts

### Reading Email
```
"List my mailboxes"
"Show me unread emails from today"
"Search for emails from john@example.com"
"Find emails about the project proposal"
"Get the full content of that email"
```

### Sending Email
```
"Send an email to boss@company.com with subject 'Status Update' and body 'All tasks complete.'"
"Forward that email to team@company.com with the note 'FYI'"
```

### Organizing
```
"Move those emails to the Archive folder"
"Mark the last 5 emails as read"
"Flag the email about the deadline"
"Add the tag 'urgent' to emails from the CEO"
"Delete the promotional emails"
```

### Multi-Account
```
"List my accounts"
"Switch to Work"
"What's the current account?"
```

## Task Prompts

### Viewing Tasks
```
"What are my incomplete tasks?"
"Show me tasks due this week"
"List all tasks including completed ones"
```

### Creating Tasks
```
"Create a task to review the Q1 report"
"Add a task 'Call dentist' due tomorrow"
"Create a high-priority task to submit the proposal by Friday"
```

### Managing Tasks
```
"Mark the dentist task as complete"
"Change the priority of the report task to 1"
"Delete the cancelled meeting prep task"
```

## Event Prompts

### Viewing Events
```
"What events do I have this week?"
"Show me tomorrow's schedule"
"List events for May 2024"
```

### Creating Events
```
"Create a meeting called 'Team Sync' tomorrow at 3pm"
"Schedule a dentist appointment for Monday at 10am"
"Add an all-day event 'Vacation' on June 15th"
"Create a recurring standup every Monday at 9am"
```

### Managing Events
```
"Reschedule my dentist appointment to next Tuesday"
"Move the team meeting to 4pm"
"Cancel the Friday standup"
"Change the location of the client meeting to Conference Room B"
```

## Workflow Examples

### Morning Review
```
"Show me unread emails from the last 24 hours"
"What tasks are due today?"
"What's on my calendar for today?"
```

### Email Triage
```
"Search for unread emails"
"Mark newsletters as read and move to Archive"
"Flag emails from my team for follow-up"
```

### Weekly Planning
```
"What events do I have this week?"
"Show me tasks due in the next 7 days"
"Create a task to prepare for the Thursday presentation"
```

### Project Cleanup
```
"Find emails about Project Alpha"
"Tag them all with 'project-alpha'"
"Move completed project emails to Archive"
```
